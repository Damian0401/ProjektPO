Lista argumentow:
1. Wysokosc mapy (liczba wieksza od 0)
2. Szerokosc mapy (liczba wieksza od 0)
3. Zakres ruchu (liczba wieksza od 0)
4. Szansa na zakazenie (liczba z zakresu 0 - 100)
5. Szansa na uleczenie (liczba z zakresu 0 - 100)
6. Poczatkowa liczba zdrowych obiektow (liczba nieujemna)
7. Poczatkowa liczba zakazonych obiektow (liczba nieujemna)
8. Poczatkowa liczba leczacych obiektow (liczba nieujemna)
9. Poczatkowa liczba lekarstw (liczba nieujemna)
10. Skalowanie mapy (liczba wieksza od 0)
11. Ilosc epok symulacji (liczba nieujemna)

Ilosc obiektow nie moze byc wieksza niz ilosc pol mapy.
